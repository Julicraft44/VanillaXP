# VanillaXP

## Add or remove xp from a specific player

### Adding XP/XP-LEVEL

- /xp (amount)L (playername)
- /xp (amount) (playername])
  
### Removing XP/XP-LEVEL

- /xp -(amount)L (playername)
- /xp -(amount) (playername)
